# Nuxt.js Coding Agent Evals

A collection of evaluations for testing coding agents and models on Nuxt.js development tasks.

## Evals

- `000-server-api-route` - Server API route implementation
- `001-routing` - Nuxt routing configuration
- `002-route-middleware` - Route middleware setup
- `003-state-composables` - State management with composables
- `004-page-meta` - Page metadata configuration
- `005-nuxt-ui-installation` - Nuxt UI installation
- `006-nuxt-ui-landing-page` - Building a landing page with Nuxt UI
- `007-nuxt-ui-form` - Form implementation with Nuxt UI
- `008-error-handling` - Error handling patterns
- `009-data-fetching` - Data fetching strategies

## Structure

Each eval contains:
- `input/` - Starting codebase
- `prompt.md` - Task instructions for the agent
- `README.md` - Eval description and details
